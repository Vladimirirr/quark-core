# Create-quarkc

通过该 CLI，您可以使用 quarkc 来开发：跨技术栈/原生Web组件或开发不依赖任何前端框架（Vue/React等）的 **独立 Web 应用(Bate)**。

- [开发独立组件](./COMPONENT.dev.md)
- [开发独立应用](./APP.dev.md)

简体中文 | [English](./README.en-US.md)

## 安装并创建

```js
npm create quarkc
```
